Introduction
Maya VRay Dome Light is a sIBL_GUI Template for Maya and VRay For Maya starting from 2K versions. It provides a bridge beetween Maya, VRay and Smart IBL. It creates a VRay Dome Light and connects the Lighting Texture to it so the Lighting is provided by the Light and not the Environment. There is also an Helper Script with everything you need to use sIBL_GUI smoothly with Maya here : sIBL_GUI For Maya - Helper Script

Options

  • Common Attributes

    Create Lights : Create the sIBL Dynamics Lights.

    Create Background : Create the Background Environment Branch ( Using the High Resolution LDR file of the current IBL Set ).

    Create Reflection : Create the Reflection Environment Branch ( Using the Reflection HDR file of the current IBL Set ).

    Create Lighting : Create the Lighting Environment Branch ( Using the Lighting HDR file of the current IBL Set ).

    Create Sun : Create the light used as Sun.

  • Additional Attributes

    Create Ground : A Plane is added to the Scene.

    Hide Lights : Hides all current Scene lights.

    Activate Linear Workflow : Set the Output Gamma to 2.2.

    Create FeedBack : Make the Feedback visible and attach to it the current IBL LDR Background image so there is a visual feedback in the Viewport.

    Feedback Radius : Sets the Feedback radius in Maya units.

  • Scene Components

    sIBL_Controls / [ Smart IBL Controls ] : There are several Controls Attributes on the sIBL_Controls Annotation available in the Channel Box.

    • Render Togglers : Activate / Deactivate the different Smart IBL Components.
    • * CC : Gamma / Gain Controls of the different Smart IBL Components.
