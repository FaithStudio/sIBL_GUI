Introduction
3dsMax MR Standard is a sIBL_GUI Template for 3dsMax and Mental Ray. It provides a bridge beetween 3dsMax, Mental Ray and Smart IBL.

Options

  • Common Attributes

    Create Lights : Create the sIBL Dynamics Lights.

    Create Background : Create the Background Environment.

    Create Reflection : Create the Reflection Environment.

    Create Lighting : Create the Lighting Environment.

    Create Sun : Create the light used as Sun.

  • Additional Attributes

    North : Orientation of the builded Environment.

    Sun Distance : Sun Distance in the Scene.
